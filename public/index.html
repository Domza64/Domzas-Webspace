Domza.xyz - This website

My personal website, I just made it for fun like all other projects... Also I think it's way cooler to have a website than just some random social media accounts, here I have the freedome to do whatever I want, however I want.

Some basic info about it: It's running on Debian Linux server, uses Nginx as proxy because same server also runs my other websites. Frontend is just simple HTML, CSS, Java Script with a little bit of Jquerry for sending requests to API on the backend that then returns some info from database.
